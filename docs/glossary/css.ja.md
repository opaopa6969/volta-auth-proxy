## これは何？

**CSSはウェブページをきれいに見せるものです。色、フォント、間隔、レイアウトを担当します。HTMLが建物の骨組みなら、CSSは塗装と装飾です。**

知らなくても大丈夫です！CSSは裏側で働いて、あなたが訪れるすべてのウェブサイトを、見苦しくなく素敵に見せています。

---

## 身近なたとえ

塗装されていない素のままの家があると想像してください（それがHTML）：

```
  CSSなし（素のHTML）：            CSSあり（スタイル付き）：

  ┌──────────────────┐             ┌──────────────────┐
  │ ようこそ           │             │ ✦ ようこそ ✦     │
  │ ここをクリック      │             │                  │
  │ テキスト           │             │ [ここをクリック]   │
  │                   │             │                  │
  │                   │             │ きれいなフォント   │
  │                   │             │ のテキスト...     │
  └──────────────────┘             └──────────────────┘
    灰色で退屈、                      色、間隔、
    全部同じ見た目                     ボタンもきれい
```

CSSはインテリアデコレーターです。「見出しを青く大きく。ボタンを中央に。段落の間にスペースを。このきれいなフォントを使って」と指示します。

## CSSはどんな見た目？

```css
/* すべての見出しを青く大きくする */
h1 {
  color: blue;
  font-size: 32px;
}

/* ボタンをきれいにする */
button {
  background-color: green;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
}

/* 段落の間にスペースを入れる */
p {
  margin-bottom: 16px;
}
```

パターンは常にこうです：

```
  誰をスタイルするか {
    何を変えるか: 新しい値;
  }
```

例えば：「すべてのボタンに対して、背景色を緑にする。」

---

## CSSはドレスコードのようなもの

CSSを学校の制服ルールだと考えてください：

- 「すべての生徒（p）は青いシャツを着ること（color: blue）」
- 「校長先生の名札（h1）は大きな文字にすること（font-size: 32px）」
- 「来客バッジ（button）は緑の背景にすること（background-color: green）」

ルールを一度書けば、ページ上のすべての該当する要素に適用されます。

---

## なぜCSSはHTMLと分かれているのか？

部屋を塗り替えるたびに壁を建て直さなければならないとしたら、ばかげていますよね！構造に触れずに塗装だけ変えたいはずです。

CSSも同じです。HTMLの構造を一切変えずに、ウェブサイトの見た目（色、フォント、レイアウト）を完全に変えることができます。だからCSSはHTMLとは別のファイルに置かれています。

## volta-auth-proxyでは

**volta-auth-proxyでは：** `volta.css`という1つのファイルが、すべてのページ（ログインページ、エラーページ、アカウントページ）の見た目を制御しています。配色やフォントを変えるには、このCSSファイル1つを編集するだけです。
