## これは何？

**HTMLはウェブページを作る言語です。ブラウザで見るすべてのものの骨組みと構造です。**

知らなくても大丈夫です！あなたが今まで訪れたすべてのウェブサイトはHTMLでできています。意識したことがなくても。

---

## 身近なたとえ

家を建てることを考えてみましょう：

```
  +--------------------------------------------------+
  |  家には構造があります：                              |
  |                                                   |
  |  - 骨組み（壁、床、屋根）      = HTML               |
  |  - 塗装と装飾                 = CSS                |
  |  - 電気と水道                 = JavaScript          |
  +--------------------------------------------------+
```

HTMLは**家の骨組み**です。「ここに見出しがあって、そこに段落があって、あそこにボタンがあって、一番下に画像がある」と定義します。見出しの色やボタンをクリックしたときに何が起こるかは言いません。それはCSSとJavaScriptの仕事です。

## HTMLはどんな見た目？

```html
<h1>お店へようこそ</h1>

<p>町で一番新鮮な果物を売っています！</p>

<ul>
  <li>りんご</li>
  <li>バナナ</li>
  <li>みかん</li>
</ul>

<button>カートに入れる</button>
```

これが作るもの：

```
  ┌─────────────────────────────────┐
  │  お店へようこそ                   │  <-- 大きな見出し
  │                                 │
  │  町で一番新鮮な果物を              │  <-- 段落
  │  売っています！                   │
  │                                 │
  │  • りんご                        │  <-- リスト
  │  • バナナ                        │
  │  • みかん                        │
  │                                 │
  │  [カートに入れる]                 │  <-- ボタン
  └─────────────────────────────────┘
```

---

## 基本パターン

HTMLは**タグ**を使います。山括弧で囲まれた言葉です：

- `<h1>...</h1>` = 大きな見出し（h1 = 見出しレベル1）
- `<p>...</p>` = テキストの段落
- `<ul>...</ul>` = リスト（ul = 順序なしリスト）
- `<li>...</li>` = リストの1項目（li = リストアイテム）
- `<button>...</button>` = クリックできるボタン
- `<img>` = 画像

ほとんどのタグはペアで使います：開始タグ`<p>`と終了タグ`</p>`。内容はその間に入れます。

ラベル付きの箱にものを入れるようなものです：

```
  [段落の始まり]
     ここにテキストがあります。
  [段落の終わり]
```

---

## なぜHTMLは重要なのか？

あなたが見るすべてのウェブページ -- Google、Amazon、銀行、このドキュメント -- はHTMLです。ブラウザ（Chrome、Safari、Firefox）がHTMLを読んで、あなたが見る視覚的なページに変換します。

ウェブページで「ソースを表示」すると（右クリック > ページのソースを表示）、HTMLが見えます。

## volta-auth-proxyでは

**volta-auth-proxyでは：** ログインページ、エラーページ、アカウントページはすべてHTMLで構築されています。ユーザー名とパスワードを求めるログインフォームが表示されるとき、それはHTMLがそのページの構造を定義しています。
