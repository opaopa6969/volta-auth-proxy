## これは何？

**Javaはプログラミング言語です。コンピュータに何をすべきか指示を書く方法です。Javaで書かれたプログラムはほぼどんなコンピュータでも動くため、世界で最も人気のある言語の1つです。**

知らなくても大丈夫です！Javaは数えきれないアプリやウェブサイトの裏側で使われています。今日もJavaで動くソフトウェアを知らないうちに使っていたかもしれません。

## 身近なたとえ

英語で料理本を書いたとします。問題は：
- フランスのシェフは読めない
- 日本のシェフは読めない
- 英語を話すシェフしか使えない

今度は、シェフの話す言語に自動的に翻訳される**ユニバーサル料理本**を書いたとします。一度書けば、世界中のどんなキッチンでも使えます。

それがJavaの大きなアイデアです：**「一度書けば、どこでも動く。」**

```
  ┌───────────────────────────────────────────┐
  │  Javaのコード                               │
  │  （一度だけ書く）                             │
  │         │                                 │
  │         ▼                                 │
  │  ┌─────────────┐                          │
  │  │ Java仮想     │  （自動翻訳機）            │
  │  │ マシン (JVM) │                          │
  │  └─────────────┘                          │
  │    │       │       │                      │
  │    ▼       ▼       ▼                      │
  │  Windows  Mac    Linux                    │
  │  ✓ 動く   ✓ 動く  ✓ 動く                   │
  └───────────────────────────────────────────┘
```

## Javaはどんな見た目？

小さなJavaプログラムです：

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("こんにちは！voltaへようこそ。");
    }
}
```

これは画面に「こんにちは！voltaへようこそ。」と表示するだけです。余分な言葉を気にしないでください。Javaは少し冗長（他の言語より言葉が多い）ですが、とても明確で整理されています。

## 「Java 21」とは？

車のモデルのように、Javaは時間とともに新しいバージョンが出ます：

```
  Java 8   (2014年) - 「定番」バージョン、今でも広く使われている
  Java 11  (2018年) - 人気の長期サポートバージョン
  Java 17  (2021年) - もう一つの長期サポートバージョン
  Java 21  (2023年) - 最新の長期サポートバージョン（voltaはこれ！）
```

バージョンごとに新しい機能と改善が追加されます。「Java 21」は「2023年にリリースされたJavaのバージョン」という意味です。「iPhone 15」と言うのと同じで、同じ製品の新しいモデルです。

## なぜJavaは人気なのか？

- **どこでも動く** -- Windows、Mac、Linux、スマートフォン、スマート家電まで
- **巨大なコミュニティ** -- 何百万人もの開発者、たくさんの無料ライブラリ
- **信頼性が高い** -- 銀行、病院、政府が重要なシステムに使っている
- **十分に高速** -- 重い処理もしっかりこなす
- **求人市場** -- 最も需要の高いプログラミング言語の1つ

---

## なぜvoltaはJavaを選んだのか？

- Javaはどのオペレーティングシステムでも動く
- Java 21にはコードをきれいで高速にするモダンな機能がある
- Javaのエコシステムには、Webサーバー、データベース、セキュリティの優れたライブラリがある
- 多くの開発者に理解されており、プロジェクトが参加しやすい

## volta-auth-proxyでは

**volta-auth-proxyでは：** サーバー全体がJava 21で書かれており、ログインリクエストの処理からセッション管理、データベースとの通信まで、すべてを担当しています。
