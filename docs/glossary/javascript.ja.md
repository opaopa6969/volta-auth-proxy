## これは何？

**JavaScriptはウェブページをインタラクティブにするものです。クリックすると動くボタン、入力をチェックするフォーム、リロードせずに更新されるページを実現します。**

知らなくても大丈夫です！JavaScriptはあなたが訪れるほぼすべてのウェブサイトの裏側で静かに動いています。

---

## 身近なたとえ

家のたとえを覚えていますか？

```
  家を建てる：

  HTML        = 壁、床、部屋（構造）
  CSS         = 塗装、壁紙、家具（見た目）
  JavaScript  = 電気、水道、ドアベル（動き）
```

JavaScriptがなければ、ウェブページは美しいポスターのようなもの。見ることはできますが、触れません。JavaScriptがものごとを「起こし」ます：

- ボタンをクリックするとメニューが現れる
- 検索ボックスに入力すると候補が出る
- フォームを入力すると送信前に「正しいメールアドレスを入力してください」と教えてくれる
- カウントダウンタイマーがリアルタイムで動く

## JavaScriptはどんな見た目？

```javascript
// 「ログイン」ボタンがクリックされたら...
button.addEventListener("click", function() {
  // ユーザー名が入力されているかチェック
  if (username.value === "") {
    alert("ユーザー名を入力してください！");
  }
});
```

細かいことは分からなくて大丈夫です！ポイントは：JavaScriptは「これが起きたら、あれをする」と指示するものです。

## JavaScriptはJavaではない

経験豊富な人でも混乱するポイントです！名前は似ていますが：

```
  Java        = サーバーやアプリを作るプログラミング言語
  JavaScript  = ウェブページをインタラクティブにするプログラミング言語

  まったく別の言語です！
```

「はし（橋）」と「はし（箸）」のようなもの。同じ読みでも、まったく違うものです。（JavaScriptは1990年代にJavaの人気に乗るために名付けられました。マーケティングです！）

---

## JavaScriptはどこで動く？

JavaScriptはウェブブラウザ（Chrome、Safari、Firefox）の中で直接動きます。ウェブページを訪れると、ブラウザは：

1. HTMLをダウンロード（構造）
2. CSSをダウンロード（見た目）
3. JavaScriptをダウンロード（動き）
4. すべてを組み合わせて、あなたが見て操作するページにする

## volta-auth-proxyでは

**volta-auth-proxyでは：** `volta.js`というファイルがログインページのインタラクティブな動作を担当しており、期限切れのセッションを自動的にリフレッシュして、ユーザーが作業を失わないようにしています。
