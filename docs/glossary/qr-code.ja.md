# QR コード

[English version](qr-code.md)

---

## 一言で言うと？

QR コードとは、スマホのカメラでスキャンするだけで URL を開いたり、データを共有したり、アクションを起こせる四角いバーコードです。入力不要。

---

## 1000回のキー入力に匹敵する1枚の画像

QR コードは、カメラだけが理解できる言語で書かれたショートカットです：

| 従来の方法 | QR コードの方法 |
|---|---|
| 「https://auth.example.com/invite/a1b2c3d4-e5f6-... にアクセスして」 | この四角をスキャン |
| 50文字の URL を電話で読み上げ | カメラを向けるだけ |
| チラシに URL を印刷しても誰もタイプしない | QR コードなら皆スキャンする |

仕組み：

- **白黒の四角**がデータ（通常は URL）をエンコード
- **角の3つの大きな四角**がカメラにコードの位置と向きを教える
- **エラー訂正**により、一部が損傷していても読み取り可能
- 標準的な QR コードは最大約4,000文字を格納できる

---

## なぜ必要なの？

QR コードがなければ：

- URL を口頭や紙で共有するとミスが起きやすい（「ハイフン？アンダースコア？」）
- モバイルユーザーが長い URL を手打ちする必要がある
- UUID 入りの招待リンクは正確にタイプするのがほぼ不可能
- 物理から デジタルへの移行（ポスター、名刺、レシート）にタイピングが必要

QR コードは、タイピングゼロで物理世界とデジタル世界を橋渡しします。

## volta-auth-proxy での QR コード

volta は**テナント招待**に QR コードを使います：

OWNER や ADMIN がテナントに誰かを招待すると、volta がこんな招待 URL を生成します：

```
https://auth.example.com/invite/a1b2c3d4-e5f6-7890-abcd-1234567890ef
```

この URL は長く、覚えられない UUID を含んでいます。volta はこれを QR コードとして提示できます：

```
  ┌──────────────────────┐
  │ ██ ▄▄▄▄▄ █ █ ██ ▄▄▄ │
  │ ██ █   █ ███ ██ █ █ │
  │ ██ █▄▄▄█ █ ███ █▄▄█ │
  │ ████████ █ █ ██████ │
  │ ██ ▄▄ ▄█ ███ █ ▄▄  │
  │ ██████ █ █ ██ █████ │
  │ ██ ▄▄▄▄▄ █ ███ ▄ █ │
  │ ██ █   █ █ █ █ ███  │
  │ ██ █▄▄▄█ ███ ██ █ █ │
  └──────────────────────┘
  スキャンで ACME Corp に参加
```

volta での使用例：

| シナリオ | QR の活用方法 |
|---|---|
| 対面オンボーディング | 画面に QR を表示、新メンバーがスマホでスキャン |
| 印刷物 | ウェルカムパケットに QR を掲載 |
| リモート招待 | メッセージアプリで QR を送信（長い URL より簡単） |
| カンファレンス/イベント | 参加者がテナントに参加するための QR を表示 |

## 具体的な例

volta テナント招待で QR コードを使う流れ：

1. 「ACME Corp」テナントの ADMIN が招待管理ページを開く
2. ADMIN が「招待を作成」をクリック
3. volta が招待 URL を生成：`https://auth.example.com/invite/a1b2c3d4-...`
4. volta がその URL をエンコードした QR コードも生成
5. ADMIN がミーティング中にノートPCの画面に QR コードを表示
6. 新しいチームメンバーがスマホのカメラを画面に向ける
7. スマホが QR コードを認識し、「[ブラウザ](browser.ja.md)で開く？」とバナーを表示
8. チームメンバーがバナーをタップ
9. [ブラウザ](browser.ja.md)が招待 URL を開く
10. 未ログインなら、volta が Google [OIDC](oidc.ja.md) 経由の[ログイン](login.ja.md)に[リダイレクト](redirect.ja.md)
11. [ログイン](login.ja.md)後、volta がユーザーを MEMBER として ACME Corp に紐づけ
12. 新メンバーが MEMBER [ロール](authentication-vs-authorization.ja.md)でテナントに参加完了

**セキュリティ注意：** QR コードのセキュリティは、エンコードされた招待 URL のセキュリティと同じです。volta の招待リンクは：

- ユニークな UUID を持つ（推測不可能）
- 1回限りまたは使用回数制限あり
- 設定時間後に期限切れ
- ADMIN が取り消し可能
